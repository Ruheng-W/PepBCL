# PepBCL: a deep learning-based model to predict protein-peptide binding residues



## How to use it

The main program in the train folder main.py file. You could change the load_config function to achieve custom training and testing, such as modifying datasets, setting hyperparameters and so on. File main.py has detail notes.



The project is mainly implemented through **Pytorch** and **sklearn**. See requirements.txt for details of dependent packages.
